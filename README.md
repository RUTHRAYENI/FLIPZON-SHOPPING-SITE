## 🛒 FLIPZON SHOPPING SITE

FLIPZON is an interactive web-based shopping application that allows users to browse products, manage cart and wishlist, and simulate an online shopping experience.
It provides features like product filtering, sorting, search, cart management, and payment simulation using a clean and responsive UI.

## 🚀 Features
- Product listing with images, price, ratings, and descriptions
- Search functionality for products
- Category-based filtering (Groceries, Electricals, Household)
- Sorting options (Price Low-High, High-Low, Name)
- Add to Cart functionality
- Add to Wishlist functionality
- Dynamic cart sidebar with total calculation
- Wishlist sidebar management
- Remove items from cart and wishlist
- Product details modal popup with reviews
- Order placement simulation
- Payment simulation with confirmation
- Dark mode toggle
- Responsive design for different devices

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage (for cart and wishlist persistence)
